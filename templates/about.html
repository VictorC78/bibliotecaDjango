SOBRE O PROJETO

Introdução

Este projeto foi desenvolvido no âmbito do curso Desenvolvimento Web com o Framework Django, oferecido pelo Instituto Federal de Educação, Ciência e Tecnologia do Sertão Pernambucano. O principal objetivo do projeto é proporcionar uma experiência prática no desenvolvimento de aplicações web utilizando Django, consolidando os conhecimentos adquiridos ao longo do curso. A iniciativa visa reforçar conceitos fundamentais como modelagem de dados, criação de rotas, manipulação de templates e integração com bancos de dados.

Descrição do Projeto

O projeto consiste em uma aplicação web que simula uma Livraria Virtual, permitindo a gestão de livros, autores, coleções e reservas. Com uma interface intuitiva e organizada, os usuários podem visualizar os livros disponíveis, consultar informações sobre autores e gerenciar suas reservas. A aplicação foi desenvolvida utilizando Django para o backend, HTML, CSS e JavaScript para a interface, e um banco de dados relacional para armazenar as informações de maneira eficiente e segura. Além disso, a plataforma conta com um sistema de autenticação de usuários, garantindo um ambiente personalizado para cada visitante.
